# YellowBlossomProject

YellowBlossomProject is a PMIS system designed for product management, similar to Jira and ClickUp.

## Folder Structure
- `doc`: Stores all documents related to the system.
- `src`: Where code is developed, tested, and stored.

## Git Branches
- `master`
- `dev`

## Technologies Used
- .NET 8.0
- React JS (not updated yet)
- PostgreSQL

## Applied Techniques
- Clean Architecture:
  - `YellowBlossom.Application`
  - `YellowBlossom.Domain`
  - `YellowBlossom.Infrastructure`
  - `YellowBlossom.Presentation`
- Repository Pattern and Dependency Injection
- Logging Implementation
- RBAC and JWT for authentication and authorization
